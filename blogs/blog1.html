CSS Box Model

The box model is a box that will wrap around every HTML element. It consists of: margins, borders, padding, and the actual content.

Let’s start with content. Content Is that part of the box where the text shows up. Next let’s talk about padding - now this is the area between the box and the content, based on what dimensions you are giving for padding, that much inner space will be created. Border- this is the outer lining around the content , after padding. Margin is like the outer space between the border and other boxes or divs. This will create an outer space like appearance .

The content area contains either text or image or video. It is the actual content.

The padding has dimensions that cover the content part .

The border has dimensions like border :size type colour.

The margin has dimensions in margin or individually margin-top, margin-bottom, margin-left, margin-right can be specified.
